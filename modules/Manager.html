Class UIElement

The element currently capturing keyboard input.
uielement:register (name, classTable)
Registers a new UI class and creates a shortcut constructor. For example, registering “Button” creates Manager.newButton().

Parameters:

  • name string The name of the class
  • classTable table The table containing the class methods

Base class for all UI components

Manager.UIElement:new (opts)
Constructor for the base UIElement.

Parameters:

  • opts Configuration options
    • id string Unique identifier for the element
    • x number X coordinate (default 1)
    • y number Y coordinate (default 1)
    • w number Width (default 18)
    • h number Height (default 12)
    • bg number Background color (default colours.blue)
    • fg number Foreground color (default colours.white)

Returns:

    UIElement

Core Loop

init (config)
Initializes the framework and starts the event listener loop. This function is blocking (yields).

Parameters:

  • config Configuration table
    • frames table List of frames to display
    • scale number Text scale for the monitor (default 0.5)
    • onUpdate function Function called every tick (0.1s) (optional)
