Module Frame

A bordered container that manages sub-windows and child elements.

Frames act as local coordinate systems for the elements inside them.

Functions

Functions

addChild (element)
Adds a child element to this frame. The element will be rendered relative to the Frame’s interior.

Parameters:

  • element UIElement The UI component to add (Button, Label, etc.)

Returns:

    UIElement The added element (allows for chaining)
click (x, y)
Handles click events and redirects them to the correct child element. Coordinates are converted to local frame space automatically.

Parameters:

  • x number The X coordinate of the click
  • y number The Y coordinate of the click
drawBorder ()
Internal method to draw the border and title.
new (opts)
Creates a new Frame instance.

Parameters:

  • opts Configuration options
    • text string The title text to display on the border (default "")
    • align string Title alignment: “left”, “center”, or “right” (default "left")
    • side string Which side the title is on: “top”, “bottom”, “left”, or “right” (default "top")

Returns:

    Frame
render ([clearBG=true])
Renders the frame and all its children.

Parameters:

  • clearBG boolean Whether to clear the background color before drawing (default true)

Internal method to initialize the ComputerCraft window object.
