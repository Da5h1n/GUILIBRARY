new (opts)
Create a new Button.

Parameters:

  • opts Options table
    • id string Unique ID (optional)
    • x number X position (default 1)
    • y number Y position (default 1)
    • w number Width (optional)
    • h number Height (optional)
    • bg number Background Color (optional)
    • fg number Foreground Color (optional)
    • mon table Target monitor (optional)
    • text string Button label (default "Button")
    • action function Callback function on click (optional)
    • bg_active number Color when pressed (default colours.white)
