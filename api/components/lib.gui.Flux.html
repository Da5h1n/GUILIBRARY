Module lib.gui.Flux

The core engine of the UI framework.

Handles element registration, event looping, and global focus management.

Functions

register (name, classTable)
Registers a new UI class and creates a shortcut constructor. For example, registering "Button" creates Flux.newButton().

Parameters:

  • name string The name of the class
  • classTable table The table containing the class methods

Base class for all UI components

UIElement
The base UI element class that all components inherit from.
UIElement
Base UI element properties. All UI components (Buttons, Labels, etc.) inherit these fields.

Fields:

  • id string Unique identifier for the element (optional)
  • x number Width (default 18)
  • y number Y coordinate (default 1)
  • x number Width (default 18)
  • h number Height (default 12)
  • bg number Background color (default colours.blue)
  • fg number Foreground color (default colours.white)
  • mon table The display target (monitor or term) (default term)
sleep (duration)
The main way to wait without blocking other UI updates.

Parameters:

  • duration number The seconds to wait before continuing.

Core Loop

init (config)
Initializes the framework and starts the event listener loop. This function is blocking (yields).

Parameters:

  • configConfiguration table
    • frames table List of frames to display
    • scale number Text scale for the monitor (default 0.5)
    • onUpdate function Function called every tick (0.1s) (optional)
