Module Button

A clickable UI component that triggers an action.

Buttons provide visual feedback by changing colours when clicked.

Class Button

button:click () Handles the click animation and executes the button’s action.
button:new (opts) Creates a new Button instance.


Class Button

button:click ()
Handles the click animation and executes the button’s action. This function is called automatically by the Manager.
button:new (opts)
Creates a new Button instance.

Parameters:

  • opts Configuration options
    • text string The label displayed on the button (default "Button")
    • action function The function to execute when the button is clicked (optional)
    • fg_active number The text colour when pressed (default colours.black)

Returns:

    Button
